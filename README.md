# sql2005-bridge
NodeJS Server with SQL connection

Do you use an old SQL Server (2005)? Did you upgrade to Windows 11? Can you no longer use T-SQL?

Well this is a solution that will solve the problem. nodeJS still works with TLS 1.0 You can run this server on the same network as the SQL server.

This will allow you to send T-SQL queries through https.

Obviously this can be modified to work for your needs but the concept can keep your old servers running.
